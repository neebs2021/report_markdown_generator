# Report Markdown Generator

A lightweight, professional web application designed to generate clean, formatted markdown updates for your team. This tool helps you track assigned and completed tasks efficiently.

## Features

- **Task Management**: Easily organize tasks into "Assigned" and "Completed" categories.
- **Multiple Projects**: Manage tasks across different projects within a single interface.
- **Markdown Generation**: Automatically generates a formatted markdown report ready for sharing (e.g., in Slack, Discord, or Email).
- **Import/Export**:
  - Import existing markdown reports to continue working.
  - Load "Assigned" tasks into "Completed" for the next day's report.
  - One-click copy to clipboard.
- **Local Storage**: Data is processed entirely in your browser; no server-side storage.
- **Responsive Design**: Modern, dark-themed UI that works on desktop and mobile.

## Usage

This is a standalone web application. To use it, simply open the `index.html` file in your preferred web browser.

1. Clone or download the repository.
2. Open `index.html`.
3. Start adding projects and tasks!

## Project Structure

- `index.html`: The main entry point and structure of the app.
- `style.css`: Contains all the styling, animations, and the dark theme.
- `script.js`: Handles the application logic, state management, and markdown generation.
